# Unwind ✈️

Unwind is a Java Spring Boot based travel planner application designed to help users explore destinations, plan trips, and manage travel-related information efficiently.

Built using Spring Boot and PostgreSQL, the project follows a clean backend architecture and is structured for scalability and future feature expansion.

---

## 🚀 Tech Stack

* **Java 21**
* **Spring Boot 3.5.3**
* **Spring Web**
* **Spring Data JPA**
* **PostgreSQL**
* **Lombok**
* **Maven**

---

## ⚙️ Features

* Travel planning backend
* RESTful APIs using Spring Boot
* PostgreSQL database integration
* Ready for future integrations like:

  * Authentication & Authorization
  * Destination recommendations
  * Booking management
  * Itinerary planning
  * Redis caching
  * MongoDB support
  * Docker deployment
